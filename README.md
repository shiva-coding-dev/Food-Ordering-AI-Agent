##  Problem Statement
Ordering food online requires repetitive manual steps like searching for dishes, applying filters, selecting restaurants, choosing items, and checking prices.

## Solution
This project is an autonomous AI agent that operates directly on an Android phone and automates the complete Zomato food ordering workflow.

The agent:
- Opens Zomato app
- Searches for food
- Applies rating filters
- Selects restaurant
- Adds item to cart
- Reads and speaks the price

## Agent Workflow

1. Open Zomato app  
2. Enable location if required  
3. Search for "Biryani"  
4. Apply "Rated 4.0+" filter  
5. Select top restaurant  
6. Add Chicken Biryani (Half Plate) to cart  
7. Tells the price  

---

##  How to Run the Agent

Use MobileRun Playground and enter the following task prompt:


Go to the home screen.
Open The Zomato App.
If it asks to enable the location then turn it on.
Tap on Search Bar.
Search for "Biryani".
Click on the first option.
Wait for the screen to load.

Click on the Filters option.
Look for the Restaurant Rating section.
Click on "Rated 4.0+".
Then click on "Show results".
Wait for the screen to load.

Click on the first restaurant.
Scroll and find Chicken Biryani.
Add 1 Chicken Biryani (Half Plate).
Add item to cart.
Open cart.

Speak the price of the item.
